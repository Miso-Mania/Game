\doxysubsection{Detailed Description}
Default base class of the \mbox{\hyperlink{classbasic__json}{basic\+\_\+json}} class. 

So that the correct implementations of the copy / move ctors / assign operators of \mbox{\hyperlink{classbasic__json}{basic\+\_\+json}} do not require complex case distinctions (no base class / custom base class used as customization point), \mbox{\hyperlink{classbasic__json}{basic\+\_\+json}} always has a base class. By default, this class is used because it is empty and thus has no effect on the behavior of \mbox{\hyperlink{classbasic__json}{basic\+\_\+json}}.
